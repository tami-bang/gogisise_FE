# 기능명세서

| **기능명** | **상세 설명** | **우선순위** | **참고사항** |
| --- | --- | --- | --- |
| **시세 데이터 자동 수집** | 매일 아침 금천미트의 한우 암소 및 한돈(암퇘지) 시세 데이터를 자동으로 수집하여 저장한다. | 상 | Python 웹 크롤링 및 PostgreSQL 사용 |
| **즐겨찾기 시세 자동 조회** | 앱을 실행하면 별도의 조작 없이 즐겨찾기 해둔 관심 부위의 오늘 시세를 메인 화면에 즉시 표시한다. | 상 | 조건 검색창 및 엑셀 표 형태 완전 배제 (Zero-Touch UX) |
| **전일 대비 변동률 표시** | 메인 화면의 시세 옆에 어제 대비 가격 상승 및 하락 폭을 붉은색/푸른색 화살표(▲/▼)와 함께 표시한다. | 상 | 돋보기가 필요 없는 특대형 텍스트 적용 |
| **축종 대분류 탭 전환** | 화면 최상단에 고정된 2개의 특대형 탭([국내산 한우 암소], [국내산 한돈 암퇘지])을 터치하여 축종 카테고리를 변경한다. | 상 | 수입육 및 거세우 데이터는 노출에서 완전 제외 |
| **보관 상태 선택 (한우)** | 한우 탭 화면 내에 위치한 대형 토글 버튼([냉장], [냉동])을 터치하여 하위 부위 목록을 전환한다. | 상 | 한돈(암)은 냉장 품목 7개뿐이므로 해당 토글 생략 |
| **관심 부위 즐겨찾기 등록** | 화면 전체를 채우는 바둑판 형태의 거대한 부위 타일(예: [삼겹], [등심])을 터치하여 즐겨찾기에 추가한다. | 상 | 텍스트 입력(자판) 방식의 검색창 일절 배제 (오타 스트레스 원천 차단) |
| **상세 시세 그래프 조회** | 메인 화면에 띄워진 특정 부위 시세 카드를 터치하면 해당 부위의 시세 변동 추이 그래프를 보여준다. | 중 | 복잡한 조건 재설정 없이 터치 1회로 결과 노출 |
| **즐겨찾기 삭제** | 즐겨찾기 목록에서 특정 부위 항목을 삭제하여 더 이상 메인 화면에 노출되지 않도록 한다. | 중 | 관리 기능 최소화 |
| **카카오톡 시세 공유** | 화면 하단의 특대형 [카카오톡 나에게 보내기] 버튼을 누르면 현재 화면의 시세 정보가 카카오톡 메시지로 전송된다. | 상 | 갤러리에 저장하여 사진을 확대(Pinch-to-Zoom)해가며 찾아야 하는 불편함 완벽 해소 |
