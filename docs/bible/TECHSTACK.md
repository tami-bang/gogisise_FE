# 기술 명세서

#### 📱 프론트엔드 (모바일 웹): React

- **추천 이유**
    
    본 서비스는 **React 기반의 모바일 웹(Mobile First)** 으로 개발한다.
    
    핵심 기능이 시세 조회, 즐겨찾기, 카카오톡 공유 등으로 구성되어 있어 카메라, 블루투스, 백그라운드 서비스와 같은 네이티브 기능이 거의 필요하지 않다. 따라서 Android와 iPhone 전용 앱을 각각 개발하는 대신, **하나의 React 웹 애플리케이션으로 두 운영체제를 모두 지원**하는 것이 가장 효율적이다.
    
    UI는 **모바일 화면을 기준으로 설계**하며, 60대 정육점 사장님을 고려해 다음 원칙을 적용한다.
    
    - 특대형 글씨와 큰 버튼
    - 높은 명암 대비
    - 단순한 화면 구성
    - 터치하기 쉬운 UI
    - 스마트폰에서 앱처럼 자연스럽게 사용할 수 있는 모바일 퍼스트 디자인
    
    또한 React는 컴포넌트 기반 구조를 제공하여 공통 UI를 재사용하기 쉽고, 유지보수와 기능 확장이 용이하다. AI 코드 생성 도구와의 호환성도 뛰어나 빠른 MVP 개발에 적합하다.
    
- **구현 방식**
    - React 기반 모바일 웹 개발
    - Android Chrome, Samsung Internet 등에서 동일하게 사용 가능
    - iPhone Safari에서 동일하게 사용 가능
    - 하나의 코드베이스로 Android와 iOS 모두 지원
    - 초기 MVP는 웹으로 빠르게 검증하고, 향후 네이티브 기능이 필요한 경우 React Native 등의 앱으로 확장 가능
