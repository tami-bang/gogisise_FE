# 메인 홈 화면 명세 - 오늘의 암컷 축산물 도매 가격

## 1. 화면 개요 및 구조 규칙
- **화면 목적**: 사장님이 바쁜 아침 시장 업무 중, 앱 진입 3초 이내에 오늘의 종합 시세와 본인의 즐겨찾기 부위 등락 단가를 즉각 획득하게 만드는 대시보드 뷰
- **가격 등락 상태 시각 분기 원칙**:
  - **시세 상승 시**: 메인 브랜드 레드 컬러 계열인 `primary` / `primary-container` (#af2f1f) 색상 적용
  - **시세 하락 시**: 1번 컬러칩에 정의된 **전용 블루 컬러인 `secondary` (#006493) 색상**을 수치와 배지에 반사하여 직관적 바이너리 피드백을 구현합니다.

## 2. 컴포넌트 조립 구성
- **최상단 앱 바 (Top App Bar)**:
  - 고정 레이아웃 구조, 🏪 상점 모양 아이콘 탑재
  - 메인 타이틀 텍스트: "오늘의 암컷 축산물 도매 가격입니다" (`headline-md`, 메인 브랜드 색상 강조)
- **상단 40% 요약 영역 (Viewing Area - Upper 40%)**:
  - 전일 대비 시장 흐름의 총평을 완성하는 대형 대시보드형 텍스트 존 구성
  - 메인 레이블: "축산 시장 종합" (`price-display`에 상응하는 거대 볼드 위계)
  - 등락 종합 지표 배지: 상승 스케일 발생 시 붉은색 트렌드 아이콘과 함께 "0.8% 상승" 안내문 노출
  - **축종별 요약 카드 미니 그리드 (Summary Stats)**:
    - 한우 암컷 스택: 우측에 등락 수치 `+1,200` 형태로 붉은색 강조 노출
    - 한돈 암컷 스택: **하락 상태이므로 전용 블루(#006493) 색상**을 받아 `-350` 수치로 푸른색 대비 노출
- **하단 60% 조작 영역 (Interaction Area - Lower 60%)**:
  - **즐겨찾는 암컷 시세 카드 리스트**:
    - 사장님이 설정해 둔 카드가 세로 단일 열 구조로 정렬 노출
    - **샘플 카드 1 (한우 암컷 등심)**: 단가 수치 21,500원을 가장 거대한 볼드로 노출하고, 우측에 상승 붉은색 배지(`▲ 1,200`) 결합
    - **샘플 카드 2 (한돈 암컷 삼겹살)**: 단가 수치 18,200원을 노출하고, **하락 상태이므로 우측에 푸른색 변동 배지(`▼ 350`) 결합**
  - **시세 추이 그래프 부품 (Price Trend Card)**:
    - 7일간의 동향을 단순화된 세로 막대 컴포넌트로 표현하되, 노안 사용자의 시각 인지 방해 요소인 미세 격자선과 자잘한 축 수치 텍스트는 전면 제거합니다.
  - **카카오톡 전송 버튼 (Sharing Button)**:
    - 엄지손가락 범위 최적화 배치, 카카오 브랜드 노란색 배경 레이아웃 위에 대형 말풍선 아이콘과 결합하여 `"카카오톡으로 오늘 시세 보내기"` 명확한 동사형 액션 레이블 매핑
- **영구 하단 앵커 내비게이션 (Bottom Navigation Bar)**:
  - 고정 높이 72px 규격 사수, 상단에 명확한 `outline-variant` 실선 구분 배치
  - 대형 아이콘과 텍스트 레이블이 수직으로 적층된 3개 탭 링크 구조 (`홈`[활성화 레드 색상 반전] / `즐겨찾기` / `설정`)
