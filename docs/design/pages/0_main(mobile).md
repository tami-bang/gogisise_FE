# 메인 홈 화면 명세 - 오늘의 암컷 축산물 도매 가격

## 1. 화면 개요 및 구조 규칙
- **화면 목적**: 사장님이 바쁜 아침 시장 업무 중, 앱 진입 3초 이내에 오늘의 종합 시세와 본인의 즐겨찾기 부위 등락 단가를 즉각 획득하게 만드는 대시보드 뷰
- **가격 등락 상태 시각 분기 원칙**:
  - **시세 상승 시**: 메인 브랜드의 상승 컬러 규칙에 맞춰 `--color-text-red` (#d14734) 색상 및 ▲ 기호 바인딩
  - **시세 하락 시**: `--color-secondary` (#3b91c8) 색상 및 ▼ 기호를 수치와 배지에 반사하여 직관적 피드백 구현

## 2. 컴포넌트 조립 구성

### 2.1 최상단 앱 바 (Top App Bar) - [Header 영역]
- **동작**: 상단 영구 고정 레이아웃 구조 (`fixed`, z-index: 50, 최소 높이: 72px)
- **구성**: 🏪 상점 모양 아이콘 탑재, 메인 타이틀 텍스트: "오늘의 암컷 축산물 도매 가격입니다" (`text-title-xl` 24px, 700, `--text-strong` #1f1f1f 강조)

### 2.2 대시보드 및 리스트 컨텐츠 - [Body 영역]
- **동작**: 내부 수직 스크롤 활성화 (`overflow-y-auto`), 좌우 패딩 20px 사수, 하단 여백 padding-bottom: 96px 필수 확보
- **상단 40% 요약 섹션 (Summary Stats)**:
  - 메인 레이블: "축산 시장 종합" (`text-display` 32px, 800, `--text-strong` #1f1f1f 적용)
  - 등락 종합 지표 배지: 상승 스케일 발생 시 붉은색 트렌드 아이콘과 함께 "0.8% 상승" 안내문 노출 (`--color-text-red` #d14734)
  - **축종별 요약 카드 미니 그리드**:
    - 한우 암컷 스택: 우측에 등락 수치 `+1,200` 형태로 상승 붉은색 강조 노출 (`--color-text-red` #d14734)
    - 한돈 암컷 스택: 하락 상태이므로 전용 경쾌한 파랑 색상을 받아 `-350` 수치로 푸른색 대비 노출 (`--color-secondary` #3b91c8)

- **하단 60% 조작 영역 (Interaction Area)**:
  - **즐겨찾는 암컷 시세 카드 리스트**:
    - 사장님이 설정해 둔 카드가 세로 단일 열 구조로 정렬 노출 (`radius-xl` 20px, `shadow-soft` 적용)
    - **샘플 카드 1 (한우 암컷 등심)**: 단가 수치 21,500원을 가장 거대한 볼드로 노출(`text-display` 32px, 800), 우측에 상승 붉은색 배지(`▲ 1,200`, 글자 `--color-text-red` #d14734, 배경 #ffedea) 결합
    - **샘플 카드 2 (한돈 암컷 삼겹살)**: 단가 수치 18,200원을 노출하고, 하락 상태이므로 우측에 푸른색 변동 배지(`▼ 350`, 글자 `--color-secondary` #3b91c8, 배경 #edf6fc) 결합
  - **카카오톡 전송 버튼 (Sharing Button)**:
    - 엄지손가락 범위 최적화 배치, 카카오 브랜드 노란색 배경 레이아웃(`--color-point-yellow` #ffc700) 위에 대형 말풍선 아이콘과 결합하여 `"카카오톡으로 오늘 시세 보내기"` 명확한 동사형 액션 레이블 매핑 (`text-body-lg` 20px, 600)

### 2.3 영구 하단 앵커 내비게이션 (Bottom Navigation Bar) - [Footer 영역]
- **동작**: 화면 최하단 상시 고정 배치 (`fixed`, z-index: 60, 기본 높이: 80px, `shadow-footer` 적용)
- **구성**: 대형 아이콘과 텍스트 레이블이 수직으로 적층된 3개 탭 링크 구조 (`홈`[활성화 레드 색상 `--color-text-red` #d14734 반전] / `즐겨찾기` / `설정`)
