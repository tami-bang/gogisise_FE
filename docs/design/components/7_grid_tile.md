# 컴포넌트 명세 - [7] 바둑판 형태의 대형 부위 선택 타일 (Grid Tile)

## 1. 컴포넌트 구조 규격
- **그리드 레이아웃 (Grid System)**:
  - 구조: 가로 스크롤 및 입력창 자판 오타 스트레스를 배제하기 위해, 화면 전체 너비를 채우는 **가로 2열(2-Column) 또는 3열 정방형 바둑판 격자** 구조 강제
  - 간격: 타일 컴포넌트 간 상하좌우 마진은 `BASE.md` -> `space-16` (16px)으로 고정 분할
- **단일 부위 타일 (Single Tile Unit)**:
  - 구조: 가로·세로 비율 1:1에 준하는 거대한 사각형 칩 컴포넌트
  - 터치 영역: 오조작 방지를 위해 타일 하나의 최소 크기를 100px × 100px 이상으로 빌드

## 2. 인터랙션 및 토큰 상속 가이드
- **시각적 외곽선 및 음영**:
  - 기본 상태: `BASE.md` -> `--color-surface` (#ffffff) 배경 위에 `1px solid --color-border` (#e5e5e5) 외곽선 및 `shadow-soft` 음영 장착
  - 내부 서체: 타일 중앙에 부위 명칭(예: [삼겹], [등심])을 `text-title` (22px, 700) 위계로 굵게 배치
- **즐겨찾기 선택 및 등록 상태 (Active Interaction)**:
  - 타일 터치 시 컴포넌트 묵직한 안착감 효과 (`active:scale-[0.98]`) 즉시 발생
  - 선택 완료 상태: 외곽선이 즉시 `2px solid --color-primary` (#f85029)로 두꺼워지며, 타일 우측 상단에 `BASE.md` 내 `Success` (#2e7d32) 또는 브랜드 상승 레드를 활용한 원형 체크 배지(`check_circle`)가 시각적으로 결합 및 노출됨
