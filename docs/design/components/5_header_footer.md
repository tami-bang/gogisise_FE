# 컴포넌트 명세 - [5] 고정형 상하단 네비게이션 (Layout Nav)

## 1. 헤더 (Header) 영역 규격
- **동작 사양**: 바디 콘텐츠 스크롤 시 밀려나지 않도록 상단 영구 고정 고수 (`fixed` 또는 `sticky`, z-index: 50)[cite: 1]
- **높이 사양**: `BASE.md` 2장 규칙을 계승하여 최소 72px, 권장 80px 규격으로 프레임 렌더링[cite: 1]
- **내부 요소 가이드**:
  - 왼쪽 뒤로가기 액션 버튼 배치 시 최소 터치 영역 `48x48dp` 확보 및 `active:scale-[0.98]` 상속[cite: 1]
  - 중앙 화면 타이틀 문구는 고대비 블랙인 `BASE.md` -> `--text-strong` (#1f1f1f) 컬러 바인딩[cite: 1]

## 2. 푸터 (Footer) 영역 규격
- **동작 사양**: 바디 내용의 길이에 상관없이 화면 최하단에 항상 밀착 고정 (`fixed` 또는 `sticky`, z-index: 60)[cite: 1]
- **높이 사양**: 기본 앱 내비게이션 바 배치 시 `80px`, 최우선 액션(CTA) 버튼 단독 배치 시 `96px` 규격 사수[cite: 1]
- **내부 요소 가이드**:
  - 바디 콘텐츠와의 시각적 분리를 위해 상단 경계선에 명확한 `1px solid --color-divider` (#eeeeee) 실선 구분선 및 `shadow-footer` 연동[cite: 1]
  - 3개 탭 링크 구조(`홈` / `즐겨찾기` / `설정`) 수평 정렬 적층[cite: 1]
  - 탭 활성화(Active) 상태 시, 해당 아이콘과 레이블을 상승 붉은색 계열인 **`BASE.md` -> `--color-text-red` (#d14734)** 로 색채 반전 처리[cite: 1]
