# 컴포넌트 명세 - [6] 특대형 카테고리 탭 및 냉동·냉장 토글 버튼 (Toggle Tab)

## 1. 컴포넌트 구조 규격
- **축종 대분류 탭 (Top Filter Tab)**:
  - 위치: 화면 최상단 앱 바 바로 하단 배치 (`fixed`/`sticky` 영역 종속)
  - 구조: 가로 너비를 50%씩 균등 분할하는 2단 탭 구조
  - 높이: 최소 64px 확보 (60대 사장님 타깃 터치 영역 보장)
- **보관 상태 토글 버튼 (Storage Toggle)**:
  - 위치: 한우 암컷 탭 활성화 시 바디 영역 최상단 배치 (한돈 암컷 선택 시 스크립트로 전면 숨김 처리)
  - 구조: 좌[냉장] / 우[냉동] 형태의 일체형 대형 알약(캡슐) 버튼 구조
  - 높이: 최소 56px 규격 사수

## 2. 인터랙션 및 토큰 상속 가이드
- **축종 대분류 탭 스타일**:
  - 활성화 상태: `BASE.md` -> `--color-primary` (#f85029) 배경색 매핑, 텍스트 `--text-inverse` (#ffffff) 볼드 반전
  - 비활성화 상태: 배경색 무채색 `#ffffff` 또는 `#f5f5f5`, 텍스트 `--text-muted` (#666666)
  - 텍스트 위계: 내부 텍스트는 무조건 `text-headline` (28px) 또는 최소 `text-title` (22px) 특대형 서체 강제 적용
- **보관 상태 토글 스타일**:
  - 활성화된 칩(Chip): `BASE.md` -> `--color-primary` (#f85029) 배경색 매핑
  - 비활성화된 칩(Chip): 흰색 배경 + 회색 테두리 규칙 준수
  - 인터랙션: 선택 변경 시 화면 슬라이드 모션을 전면 금지하며, `BASE.md` 15장 규칙에 따라 150ms 내외의 부드러운 색상 전환(`duration-150`)만 적용
