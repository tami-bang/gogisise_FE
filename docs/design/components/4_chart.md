# 컴포넌트 명세 - [4] 미니멀 가격 추이 차트 (Chart)

## 1. 컴포넌트 제약 사양
- **금지 지침**: 금융형 대시보드의 미세 점선, 촘촘한 가로/세로 격자선, Y축의 자잘한 단가 수치 텍스트 전면 금지 및 제거[cite: 1]
- **허용 지침**: `BASE.md` 15장 차트 규칙에 따라, 7일간의 도매 시세 트렌드 흐름만 두터운 그래픽으로 노출[cite: 1]

## 2. 구조 및 시각화 가이드
- **가로축 레이블**: `목 | 금 | 토 | 일 | 월 | 화 | 오늘` 순으로 수평 단일 열 정렬 (`--text-muted` #666666 상속)[cite: 1]
- **차트 바디 (Bar Component)**:
  - 일주일간의 시세 편차 Mock 데이터 높낮이에 대응하는 **7개의 두터운 세로 막대(Bar) 형태**를 화면 너비에 맞춰 균등 배치[cite: 1]
  - 각 세로 막대 컴포넌트 상단 코너에 `radius-sm` (8px) 라운딩 처리[cite: 1]
- **오늘 포인트 하이라이트**:
  - 가장 우측 '오늘' 요일에 해당하는 마지막 7번째 막대는 가시성 극대화를 위해 배경색을 브랜드 메인 컬러인 **`BASE.md` -> `--color-primary` (#f85029) 주황색**으로 100% 가득 채움 처리[cite: 1]
