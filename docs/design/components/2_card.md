# 컴포넌트 명세 - [2] 카드 외곽선 및 리스트 아이템 (Card)

## 1. 컴포넌트 규격
- **배경색**: 순수 흰색 표면 (`BASE.md` -> `--color-surface` #ffffff 상속)[cite: 1]
- **모서리 둥글기**: 대형 카드의 물리적 부피감을 위해 `radius-xl` (20px) 토큰 고정 바인딩[cite: 1]
- **내부 여백**: 가시성 확보를 위해 사방 패딩 최소 `space-20` (20px) 이상 레이아웃 격자 적용[cite: 1]
- **음영 효과**: `BASE.md` -> `shadow-soft` (`0 4px 16px rgba(0,0,0,0.06)`) 상속을 통해 캔버스 배경과 명확히 분리[cite: 1]

## 2. 구조적 외곽선 및 선택 피드백 가이드
- **기본 외곽선**: 정보 경계 붕괴 방지를 위해 배경과 대조되는 **`1px solid --color-border` (#e5e5e5) 또는 `#eae7e7` 필수 렌더링**[cite: 1]
- **리스트 타깃 높이**: 세로 단일 열 정렬 시, 개별 카드의 최소 터치 유효 높이는 72px 이상 확보하여 오조작 차단[cite: 1]
- **리스트 구분선**: 카드 리스트 항목 사이에는 `2px` 두께의 명확한 실선 구분선(`--color-divider` #eeeeee) 배치[cite: 1]
- **선택 완료 상태 (Active Card)**:
  - 테두리: 즉시 **`2px solid --color-primary` (#f85029)** 로 확장 및 두꺼워짐[cite: 1]
  - 배경: 주황 연한 틴트 테두리 배경색(`#ffedea`)으로 반전되어 강력한 시각적 안착감 제공[cite: 1]
