사용자가 채팅창에 `/structure`를 입력하면, 에이전트는 프로젝트 내부 파일들을 개별적으로 읽어서 분석하지 말고(토큰 낭비 방지), 반드시 아래의 2단계 워크플로우를 직접 실행하여 메모리를 갱신해야 한다.

## [실행 프로세스]

### 1단계: Tree Skill 실행 (디렉토리 구조 파악)
현재 OS 환경(macOS)에 맞는 터미널 명령어를 직접 실행하여 최상위 디렉토리 구조를 한눈에 파악한다.
- **실행할 명령어:** ```bash
  tree -L 4 -I "node_modules|.git" --dirsfirst
